🔑 ENVIRONMENT VARIABLES (Required)

Add these in Netlify Dashboard:
Site settings → Environment variables → Add variable

Variable 1:
  Name:  VITE_SUPABASE_URL
  Value: [Get from Supabase Dashboard → Settings → API]

Variable 2:
  Name:  VITE_SUPABASE_ANON_KEY
  Value: [Get from Supabase Dashboard → Settings → API → anon/public key]

Variable 3:
  Name:  VITE_CLERK_PUBLISHABLE_KEY
  Value: [Get from Clerk Dashboard → API Keys → Publishable key]

Variable 4:
  Name:  GEMINI_API_KEY
  Value: [YOUR ACTUAL API KEY - NOT PLACEHOLDER!]
         Get from: https://makersuite.google.com/app/apikey
