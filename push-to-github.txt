# STEP 1: Replace USERNAME with your actual GitHub username
# Example: If your GitHub username is "johndoe", change:

# STEP 2: Run these commands (after replacing USERNAME):

git branch -M main
git push -u origin main

git branch -M main
git push -u origin main
